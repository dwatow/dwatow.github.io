    ¶Preset

    Plugins · Babel 頁面可以看到簡單的介紹

    在此官網開頭就解釋[2] (看原文會更好)

    Babel is a compiler, 一種吃 code 吐 code 的東西。
    將程式碼解析之後，再吐出來，並沒有做任何的處理。
    想要處理，要給 plugin。

    其中 preset 就是一種 plubin
    而 env 是一種 preset
