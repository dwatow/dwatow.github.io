不應該過渡依賴全域變數，應該要將該變數變成 class 裡的變數，並且若需要值的傳遞時，必須要透過 method 進行。 這樣 trace bug 也直覺，也更好防堵 bug 產生。 無言的聲音: 浪費了 c# 的物件導向…微軟的工程師，辛苦了…你們的辛苦寫的程式碼，完無沒有體現價值。
