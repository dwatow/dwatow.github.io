剛學 OO 的朋友，會覺得繼承超棒的，可以共用成員變數。

這其實是不好的。

剛學事件的朋友，會覺得掛上事件超棒的，可以一個 function 到處使用，這其實也是不好的。
剛學泛型的朋友，會覺得泛型寫法超棒的，可以一個 function 所有物件都可以使用，這其實也是不好的。
這些技術存在著這些偏見式的用法，卻造成語意不正確的情況 開始產生了一些，讀起來不知所云的寫法。
