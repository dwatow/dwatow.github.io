    7. Hexo - sel code block line number #2612

      參與 Open Source Project

      選用 hexo 的原因，是因為它是用 JavaScript 實作而成的工具，未來有一天可以改它。想不到來得很早，當初的選擇是對的。
      自己是 hackmd 的愛用者， hexo 剛好也使用 markdown 撰寫，所以就想把 hackmd 當作編輯環境。
      hexo 設計「文章中的程式碼區塊有無行號」的設定方式，以往只能在 config 檔設定，hackmd 可以每個區塊使用語法設定。
      就將 hackmd 的這個功能移植到 hexo 上。

      過程看了 hexo 的 source code，也看了 hackmd 的 source code。也問了 hackmd 的作者，感謝他的細心引導讓我可以順利的找到我想找的程式碼。
