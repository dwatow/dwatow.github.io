¶心得

Model 主要還是 JS 餵進去的資料為新增時回傳的物件。
並不會依照資料庫 (或 Model) 的預設值而自動幫你加上去，id 也是
除非在新增完之後，再從資料庫撈一次，該在資料表上的內容就會出現在 JS 物件中了。

而 sequelize 在 Model 上可以設定 DefaultValue 的設定，是為了搭配 .sync() 吧？從 Model 新增 Table 的方式。
