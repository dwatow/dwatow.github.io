    移植時的最後更新日期: 2012-11-13T10:48:11.321+08:00

    這堂課要和佛教的四聖諦「苦、疾、滅、道」有關…（這是上課老師預防我睡著講的） 


    有系統的理性思考
    SA - Situation Appraisal 狀況評估
    PA - Problem Analysis 問題分析
    PPA/POA - Potential Problem/ Opportunity Analysis 潛在問題/ 機會分析
    DA - Decision Analysis 決策分析
    CA - Competitiveness Analysis 競爭分析



    1. SA - Situation Appraisal 狀況評估
    A system process for prioritizing concerns 優先順序先決


        P - Plan and Identify Concerns 計畫，找出問題
        D - Do Priority Setting 優先權設定
        C - Check Process Characteristics 處理的特性
        A - Action for concerns 執行特性


        重點：定義門問題點（可能不只一個）再依輕重緩急排序
        訣竅：先抓大項，再找小項。



    2. PA - Problem Analysis 問題分析
    A system process for finding the cause of a negative (or positive) deviation

    使用時機：Deviation（與預期偏差）並找出原因，採取對策（治標？治本？）

        P - Descibe Problem 說明問題
        D - Identify Possible Cause 辨別可能的原因
        C - Evaluate Possible Cause 評價可能的原因
        A - Confirm True Cause 確認真正的原因


        P

    		 is（顯性） is not（隱性）
    Who 人*

    How 事*

    When 時

    Where 地
    What 物

    		→醫生問診的方式
                *:Extent
        D
            魚骨圖或歸納法都可以（回到問診單刪去法）
        C
            推論原因MPC（Most Probable Cause）
        A
            實驗真因（設計實驗/田口方法）
            （還原現場）



    3. PPA/POA - Potential Problem/ Opportunity Analysis 潛在問題/ 機會分析
    A systematic process for protecting future plans proactively

    使用積極計劃，保護未來


        PPA是PA的積極動作：
        PA：解決過去已發生的的問題
        PPA：預防未來可能發生的問題

        首部曲：
            1. Identigy Potential Problem
            2. Identify Likely Causes

        二部曲：
            3. Take Preventive Action
            4. Contingent Action and Triggers

        首部曲和二部曲無關聯性唷



    4. DA - Decision Analysis 決策分析
    A system process for making a choice transarently（透明）


        P - Plan Objectives 規格
        D - Develop Alternatives 選擇主次要
        C - Check Unique Risks/ Benefits 獨特性/效益
        A - Action for Selection

        P
            開出比較條件
                1. Must 絕對必須→基本門檻
                2. Want 相對必須→給權重，做比較
        D
            Must
                有→留下
                無→刪去
            Want
                給權重→給分數→算分數→分數較高者（群）留下
        C
            可PPA/POA
            加分、扣分、增加差異性（獨特性）
        A
            執行



    5. CA - Competitiveness Analysis 競爭分析A systematic process for understanding competion（了解競爭）

        1. 買家、供應商、現況、對手、潛在、替代…等的能量
        2. 位置（市場）
        3. 影響"blance" 納許均衡
        4. 利用change（改變實力為最高指導原則）
