去除外來指定元素

void removeMember(const CTest& objTest)
{
    result.erase(remove(result.begin(), result.end(), objTest), result.end());
}
上例的vector是裝string，如果是裝自訂的類別CTest，則需要重載運算子。
sort()需要重載<運算子。
unique()需要重載==運算子。
