¶誤解

以下都是誤解

  1. JS是一種直譯式語言，不會經過編譯。
  2. this關鍵字是參考所有在的函式
  3. type null 是 null
  4. 弱型別，就是沒有型別
  5. javascript的類別、繼承…(物件導向特性)
  6. 想要完整學會javascript

粗略的正確的情況:

  1. JS會在即將使用前，會先編譯。
  2. this是「誰呼叫函式」
  3. type null 是 object
  4. 是「值型別」，無「變數型別」，typeof() 是「該變數的值型別」
  5. javascript的行為委派
  6. javascript還在持續發展中，永遠都有未知的部份
