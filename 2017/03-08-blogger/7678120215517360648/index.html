使用div把內文包進來，不過希望內文有多欄(沒錯！像Word這樣)式排版的話，有語法唷！設定一下就可以了。

在此做法中，放置文字的區塊高度無須訂值。它會自動將內容平均分擔在每一欄裡面

CSS

主要在css加上 
.multi_column {
    column-count: 2;
    column-gap: 50px;
}

  • column-count 欄位數量, 此例為兩欄。
  • column-gap 設定欄間隔空白大小。

HTML

在HTML中加上 
<div class=“multi_column”>寫了很多很多內容</div>
