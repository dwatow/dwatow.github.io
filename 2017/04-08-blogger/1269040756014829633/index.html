情境


前端工程的路，剛開始常遇到「網頁寫好了，不知道RWD跑得如何」。
如何在不上傳架站空間、雲端開發環境的情況之下，進行真實設備(手機、平板)的測試呢？

解決方式


使用Linux內建的python裡的web server。


在你的index.html的目錄之下。
開啟Terminal。

確認python版本

(參數是大寫V)
$ python -V
我的ubuntu顯示如下 
Python 2.7.12

建立臨時的web server

指定port(在此例port用8000)。[1] 
$ python -m SimpleHTTPServer 8000
然後開啟你的設備，打開你的IP:8000[2]
然後就可以看見你寫的網站首頁在你自己的設備執行，進行快速簡易的真實測試囉！

注意事項


由於Linux內建python，所以可以直接這樣下指令。
若是在windows的話，就要另外下載python(不限定版本，不過要查一下指令是否通用)。

python 3.6.1版的指令如下(不一樣囉)[3]
$ python -m http.server 8000
