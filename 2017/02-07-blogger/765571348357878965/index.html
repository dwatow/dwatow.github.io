安裝Java[1]


安裝jre


直接安裝最新版7(如果有更新，此指令就要換囉~) 
$sudo apt-get install openjdk-7-jre

檢查是否安裝成功，看版本號碼 
$java -version

指定更換預設執行的版本 
$sudo update-alternatives --config java

再輸入你要使用的版本索引號碼。
更換完畢後檢查一下版本(前面有講過)

安裝jdk


直接安裝最新版7(如果有更新，此指令就要換囉~) 
$sudo apt-get install openjdk-7-jdk

檢查是否安裝成功，看版本號碼 
$javac -version

指定更換預設執行的版本 
$sudo update-alternatives --config javac

再輸入你要使用的版本索引號碼。
更換完畢後檢查一下版本(前面有講過)


安裝錯了！


連同參數設定一起移除(較乾淨)
$sudo apt-get --purge remove openjdk-7-jdk
成功之後，再重新安裝囉~


下載Jenkins[2][3]


將Jenkins的Debian軟體自動安裝包與升級，加到你的系統上
$wget -q -O - https://pkg.jenkins.io/debian/jenkins.io.key | sudo apt-key add -

在/etc/apt/sources.list加上這一行
deb https://pkg.jenkins.io/debian binary/

更新你的local package index，最後安裝Jenkins
$sudo apt-get update
$sudo apt-get install jenkins

設定開機啟動


由於目前我們的Server並沒有灌Web Service(tomcat之類的)所以，使用Jenkins自帶的就好。

編輯/etc/rc.local 加上 
java -jar /usr/share/jenkins/jenkins.war --httpPort=12345 --daemon --logfile=/home/username/.jenkins/jenkins.log
