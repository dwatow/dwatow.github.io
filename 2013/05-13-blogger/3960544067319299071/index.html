Ubuntu//網路芳鄰的功能(samba)

¶Ubuntu//網路芳鄰的功能(samba)

原文連結: https://darkblack01.blogspot.com/2013/05/ubuntu_13.html
移植時的最後更新日期: 2014-09-20T11:59:14.888+08:00

windows中常見的網芳功能，在Ubuntu可真是…（我覺得滿珍貴的哈）

還好找到了教學[1]，教你怎麼做到這件事，而且還真的不是一件簡單的事呢！難怪windows肥大和方便。



步驟如下：

1. 看看有沒有安裝samba
$ sudo dpkg -l samba*

2. 更新apt-get
$ sudo apt-get update

3. 安裝 samba
$ sudo apt-get install samba

4. 備份原始設定檔
$ sudo cp /etc/samba/smb.conf smb.conf.bak

5. 用vim編輯設定檔
$ sudo vim /etc/samba/smb.conf

6. 進入vim


[global] 

workgroup = WORKGROUP                          <= 預設WORKGROUP,你可以改成自己喜歡的工作群組
server string = Kgs_Ubuntu <=自己電腦的名稱，自己改自己喜歡的，%h就是hostname
(不一定)
security = user                                            <= 預設要輸入帳號密碼，如果不想這麼麻煩可以改成share…

[SHARE]
path = /home/kgs/wiki;     要分享的路徑  
browseable = yes ;          是否可瀏覽
read only = no ;               是否唯讀
(非必要)
create mask = 0644 ;檔案遮罩
directory mask = 0755 ;資料夾遮罩

7. 編輯完成用「:x」

8. 設定samba的使用者密碼
$ smbpasswd -a xxxx(xxxx為使用者名稱)
New SMB password:
Retype new SMB password:
Added user xxxx
成功了！
試試看用windows連Ubuntu吧！
