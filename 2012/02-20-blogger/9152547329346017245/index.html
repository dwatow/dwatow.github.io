在C++中
this是指向自己的指標
this是物件主體
（引用《指標的藝術》的隱喻this，就是拿鑰匙(*)打開信箱(point)，看見物件主體）



class MyCls
{
    MyCls  returnCopy(){（Something Coding）    return *this};
    MyCls& returnMyself(){（Something Coding） return this};
    MyCls returnPoint(){（Something Coding）   return  this};
};
