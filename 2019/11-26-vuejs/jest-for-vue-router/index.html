¶new VueRouter() 物件，要長在 LocalVue 上面

$route 和 $router 兩個會在 localVue 上成為唯讀的 property (屬性)，不可以用 mock 來覆寫。

¶避免安裝全域的 Vue Router

最好是每次測試，每次都安裝足夠用的 router 就可以了。
