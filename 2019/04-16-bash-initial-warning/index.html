¶bash 開啟出現錯誤訊息

¶問題描述

開啟 bash 時，初始化會出現錯誤訊息

¶解決方式

有些解決方式建議重裝 bash ，但是已試過無用。

重新安裝 gawk[1] (GNU awk)

竟然就好了。
